# Changelog

All notable changes to Prompt Enhancer will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-28

### Added
- Initial release of Prompt Enhancer Chrome Extension
- Manifest V3 Chrome Extension architecture
- Content scripts for ChatGPT, Claude, and Gemini
- Core enhancement engine with auto-category detection
- 3 enhancement levels: Light, Medium, Aggressive
- 8 pre-built templates: General, Coding, Writing, Analysis, Creative, Learning, Academic, Business
- Floating "✨ Enhance" button injected on supported AI platforms
- Right-click template/level dropdown on the enhance button
- Premium dark glassmorphism popup UI
- Quick enhance textarea in popup
- Enhancement history (last 50 entries)
- Hybrid mode: template-based (offline) + optional AI API
- Gemini API integration for AI-powered enhancement
- OpenAI API integration for AI-powered enhancement
- Options page with API key management
- Default template and level preferences
- History export (JSON) and clear functionality
- Toast notifications for user feedback
- Keyboard shortcut (Ctrl+Enter) for quick enhance
